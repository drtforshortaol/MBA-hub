Troubleshooting

If this app shows old information or does not load correctly, try the steps below.

Clear cache / reload app

Use the Clear Cache / Refresh App button near the top of this page. The app will remove stored files and reload fresh content.

PC manual refresh

On a PC, press Ctrl + Shift + R to force a hard refresh.

iPhone Safari cache clearing
  1. Open Settings.
  2. Go to Apps.
  3. Open Safari.
  4. Go to Advanced, then Website Data.
  5. Use the search bar and type git if the app is hosted through GitHub.
  6. Delete matching website data, then reopen the app.
