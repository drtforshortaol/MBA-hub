# Kelp Forest

## Purpose

This app is a Monterey Bay Aquarium volunteer companion page for the Kelp Forest exhibit.

It provides guide-friendly quick facts, visitor questions, conservation messages, story prompts, references, related tags, and troubleshooting support.

## Intended Users

- Monterey Bay Aquarium volunteers
- Guide trainers
- Visitor-facing educators using internal reference material

## Main Features

- PWA-ready structure
- Mobile-first design
- Dropdown content sections
- Search box
- Jump-to-section menu
- Return to MBA Hub button
- Clear Cache / Refresh App button
- Troubleshooting dropdown
- Notification area
- References and Sources section
- Tags and related items
- Offline caching through service worker

## How to Use

1. Open `index.html` in the app folder or from the Hub link.
2. Use the dropdown sections to review content.
3. Use the search box to find specific words or topics.
4. Use the jump menu to move directly to a section.
5. Use Clear Cache / Refresh App after updates.
6. Use Troubleshooting if the app shows old content or does not load correctly.
