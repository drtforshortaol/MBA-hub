# Five Senses Guide

Version: v1.1  
Last Updated: 2026-06-28  
Folder: `apps/five-senses-pwa/`

## Purpose

Five Senses Guide is a compact Progressive Web App for learning about sight, hearing, smell, taste, and touch.

## Intended Users

- Students
- Families
- Teachers
- Tutors
- General learners
- Hub users who need a simple educational dropdown guide

## Main Features

- Mobile-first layout for iPhone Safari
- Desktop-friendly layout for PC Chrome
- Standard top control bar:
  - Return to Hub
  - Clear Cache
  - Troubleshooting
- Dropdown-first content structure
- Search across sense sections and learning sections
- Expand All, Collapse All, and Reset Search controls
- Featured visual header image
- References and image notes
- Version display
- Offline support through service worker caching
- PWA manifest and install support
- Accessible button sizing, readable text, alt text, and high-contrast colors
