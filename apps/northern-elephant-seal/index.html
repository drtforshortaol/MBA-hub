Troubleshooting

Fast fixes

  • Old version showing: tap Clear Cache, then reload.
  • PC hard refresh: press Ctrl + Shift + R.
  • Images missing: open online once so the images can cache, then test offline again.
  • Offline not working: reload once while online, then reopen the app.
  • Return to Hub broken: use the browser back button and check the app folder path.

iPhone Safari cache clearing

  1. Open Settings.
  2. Go to Apps or Safari.
  3. Open Advanced if needed.
  4. Open Website Data.
  5. Search for the site, GitHub, or app keyword.
  6. Delete matching website data.
  7. Reopen the app.

This panel closes with the × Close button, by tapping Troubleshooting again, or with the Escape key on PC.
