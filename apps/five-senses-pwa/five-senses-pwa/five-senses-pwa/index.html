    Troubleshooting

    The app still shows the old version

    Tap Clear Cache, then reload the app. On PC, you can also press Ctrl + Shift + R.

    The app looks broken

    Refresh the page. If that does not work, use Clear Cache and reopen the app.

    iPhone Safari cache clearing
    1. Open Settings.
    2. Open Safari.
    3. Go to Advanced, then Website Data.
    4. Search for the hosting site keyword, such as “git”.
    5. Delete matching website data and reopen the app.
    Offline mode does not work

    Open the app while online first. Let it fully load, then test offline again.

    Return to Hub does not work

    Use the browser back button or reopen the Hub directly. Then check whether this app is stored at apps/five-senses-pwa/ so the Hub link path works.

    Troubleshooting panel will not close

    Tap × Close. On PC, press Escape. If needed, refresh the page and then use Clear Cache.
