    Troubleshooting

    If the app shows old content

    Tap Clear Cache, then reload the page. On PC, you can also press Ctrl + Shift + R.

    If dropdowns do not open

    Reload the app. If the issue continues, clear cache and reopen from the Hub.

    If the app will not open offline

    Open it once while online, let it fully load, then test offline again.

    If the Hub button does not work

    Use the browser back button or reopen the MBA Hub directly, then report the broken link.
