# MMC Narrations

Version: v1.0  
Last updated: 2026-06-29  
Status: Draft content shell

## Purpose

MMC Narrations is a mobile-first dropdown PWA for organizing narration prompts, interpretive talking points, short scripts, visitor questions, and related source notes.

## Intended Users

Monterey Bay Aquarium volunteers, guides, trainers, or app maintainers who need quick mobile access to narration material.

## Main Features

- Hub-compatible top bar
- Return to MBA Hub button
- Clear Cache button
- Troubleshooting panel with working close control
- Searchable dropdown sections
- Expand All and Collapse All controls
- References section
- Version display
- PWA manifest
- Service worker for offline support after first load
- Mobile-first layout for iPhone Safari and responsive desktop layout
