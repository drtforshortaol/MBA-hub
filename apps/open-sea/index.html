Troubleshooting

```

The app still shows an older version

Select Clear Cache, then allow the app to reload.

PC Chrome hard refresh

Press Ctrl + Shift + R.

iPhone Safari cache clearing

  1. Open Settings.
  2. Open Apps, then Safari.
  3. Open Advanced and Website Data.
  4. Search for the Hub website or GitHub domain.
  5. Delete the matching stored website data.
  6. Reopen the MBA Hub.

The app will not open

Check the internet connection, return to the MBA Hub, and reopen the app. If the problem continues, clear the cache and reload.

The app looks broken

Refresh the page. If the problem continues, clear the cache and confirm that the newest version is loaded.

Search does not work

Clear the search field, select the All filter, and try again. If needed, clear the app cache and reload.

Return to the Hub

Use the ← MBA Hub button. If the button does not work, use the browser Back button or reopen the Hub directly.

Welcome to the Open Sea

```

Unlike coastal and seabed habitats, the open sea is not tied to a touchable geological surface. It is the largest ecosystem on Earth — vast and variable, encompassing everything pelagic: open-ocean waters away from coastal limits and above the seafloor.

Habitats such as rocky shores, kelp forests, and the seabed are defined by tangible features. The open sea is shaped by intangible variables — conditions that cannot be touched but strongly influence every organism living there.
