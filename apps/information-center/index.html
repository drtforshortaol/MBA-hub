Troubleshooting

Update or refresh this app

  1. Tap Clear Cache in the top bar.
  2. Let the page reload.
  3. Confirm the version shown below is current.

PC hard refresh

Press Ctrl + Shift + R in Chrome to force the browser to reload the newest files.

iPhone Safari cache

  1. Open iPhone Settings.
  2. Go to Apps → Safari → Advanced → Website Data.
  3. Search for the site, delete matching website data, then reopen the app.

If something looks broken

Clear cache first. If the problem continues, reopen the MBA Hub and launch this app again.
