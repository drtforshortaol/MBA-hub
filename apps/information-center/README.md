# Information Center Handbook — Version 2.0

Drop-in PWA replacement for the MBA Hub Information Center app.

Folder target:

```text
apps/information-center/
```

Main features:

- 18 handbook sections
- Return to Hub button
- Search with matching-section expansion and highlighting
- Quick navigation buttons
- Expand All / Collapse All
- PWA manifest
- Service worker/offline caching
- Installable on iPhone, Android, and desktop

After uploading, test from the Hub and verify the service worker cache updates. If an older iPhone copy appears, refresh Safari or remove/reinstall the Home Screen icon.
