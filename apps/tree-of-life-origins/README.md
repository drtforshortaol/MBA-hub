# Tree of Life Origins v2.0.0

## Upload location
Upload the complete folder as:

`apps/tree-of-life-origins/`

Direct URL:

`apps/tree-of-life-origins/index.html`

## MBA Hub registry
Copy the object from `hub-registry-entry.js` into the main array in your existing `hub-registry.js`.

## Replacement procedure
1. Delete or rename the existing `apps/tree-of-life-origins/` folder.
2. Upload this complete `tree-of-life-origins` folder.
3. Replace the old registry object with the supplied v2.0.0 object.
4. Use the MBA Hub Clear Cache control.
5. Reload the Hub and open Tree of Life Origins.
6. Open the app once online before testing offline.
