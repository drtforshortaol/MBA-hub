References and Notes

Hub categories, app registry structure, tags, testing expectations, troubleshooting, and revision standards are based on the MBA Hub Core Categories Version 2 protocol.

Hub-wide app listings are maintained in hub-registry.js. Hub-wide tag definitions are maintained in hub-tags.js. Optional intentional cross-links are maintained in hub-links.js.

Individual app content should remain inside each app folder. Individual app data.js files are for that app’s own internal content, not for the root Hub registry.

Troubleshooting

Refresh Hub 2

  1. Tap Clear Cache.
  2. Let Hub 2 reload.
  3. Confirm the version at the bottom shows the newest date.

PC Chrome Hard Refresh

Press Ctrl + Shift + R to force Chrome to reload the newest Hub files.

iPhone Safari Cache

  1. Open iPhone Settings.
  2. Go to Apps → Safari → Advanced → Website Data.
  3. Search for the site, delete matching website data, then reopen Hub 2.

Broken App Link

If an app opens incorrectly or Return to Hub gives 404, check that the app folder is inside apps/app-folder-name/ and that the app uses ../../index.html for the MBA Hub button.

App Still Shows Old Version

Use Clear Cache, then reload. If the old version still appears, close and reopen the browser.

On iPhone, remove the old Home Screen icon if needed and add the Hub again after clearing Safari website data.

Service Worker Blocking Updates

If the live Hub does not match the files you uploaded, the browser may be showing old cached files.

On PC Chrome, open Developer Tools, go to Application → Service Workers, unregister the old worker, then clear site data and reload.

ChatGPT Recents Missing in Chrome

If ChatGPT Recents or conversation history are missing in Chrome but still appear on your iPhone, the chats are probably still saved to your account.

The issue is usually Chrome cache, cookies, site data, extensions, or temporary sync.

  1. Hard refresh: Press Ctrl + Shift + R.
  2. Sign out and back in to ChatGPT.
  3. Clear only ChatGPT site data: open chrome://settings/content/all, search chatgpt.com, delete the site data, reload ChatGPT, then sign in again.
  4. Restart Chrome completely.
  5. Restart your PC if needed.

Check what you are seeing: blank left sidebar, “Unable to load history,” spinning/loading indicator, or missing sidebar.

Tip: A screenshot usually helps diagnose the problem faster.

Tags and Cross-Links

Tags should appear on each app card. Clicking a tag should filter the Hub search to that tag.

Related app links can come from two places: relatedApps in hub-registry.js, or automatic shared-tag matching between apps.

Intentional special cross-links can be stored in hub-links.js.

Hub Registry Safety

Root Hub registry files now use unique names: hub-registry.js, hub-tags.js, and hub-links.js.

Individual app content files may still be named data.js, but they should stay inside the individual app folder.
